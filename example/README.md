# FaceHash Example

Example app for [facehash](https://pub.dev/packages/facehash) — a Flutter port of [facehash.dev](https://www.facehash.dev/).

This entire Flutter package was built with [Claude Code](https://claude.ai/code).

## Running

```bash
cd example
flutter run
```

## Sections

- **try it** — Interactive demo with text input, shape selector, sizes, and variant toggles
- **props** — Visual reference for every prop: `name`, `colors`, `intensity3d`, `variant`, `showInitial`, `enableBlink`, `mouthBuilder`
- **use cases** — Chat messages and avatar stack layouts
